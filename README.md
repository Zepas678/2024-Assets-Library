# 🏛️ The School of Oblivion - 3D Assets Library 

Este repositorio contiene la biblioteca completa de modelos 3D, texturas y recursos visuales desarrollados para el proyecto **The School of Oblivion**. El objetivo de este espacio es centralizar el arte del juego, asegurando una correcta gestión de mallas y materiales.

## 🎨 Contenido del Repositorio
* **Personajes:** Modelos base, skins y mallas con Rigging (FBX/Blend).
* **Escenarios:** Props de entorno, estructuras modulares y elementos de ambientación.
* **Texturas:** Mapas de Albedo, Normales y Metálicos optimizados para Unity/URP.
* **Animaciones:** Archivos de movimiento exportados para integración directa.

## 🛠️ Especificaciones Técnicas
* **Software Principal:** Blender.
* **Formatos:** `.fbx` para integración, `.blend` para edición y `.png` para texturas.
* **Optimización:** Algunos modelos con topología limpia y uso eficiente de UVs para reducir el consumo de memoria de video (VRAM).

## 📂 Organización de Carpetas
* `/Characters`: Modelos de protagonistas y enemigos.
* `/Environment`: Paredes, Mesas y Tableros.
* `/Textures`: Atlas de texturas y materiales compartidos.
* `/Animations`: Clips de animación por separado.
